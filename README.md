# TC2_2026

Repositorio de trabajos prácticos y notebooks de **Teoría de Circuitos II**.

## Contenido

- Análisis nodal de circuitos
- Obtención analítica de funciones de transferencia \(H(s)\)
- Normalización y desnormalización
- Diseño de filtros activos
- Simulaciones en LTspice
- Validación simbólica en Python con:
  - SymPy
  - Lcapy
  - pytc2

## Notebooks incluidos

- `pasa_todo_primer_orden.ipynb`
- `pasa_todo_segundo_orden.ipynb`

## Herramientas utilizadas

- Python
- Jupyter Notebook
- LTspice
- SymPy
- Lcapy
- pytc2

## Objetivo

Documentar el desarrollo teórico y práctico de circuitos vistos en la materia, incluyendo análisis matemático, simulación y verificación de resultados.
